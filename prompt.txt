### dockerfile
```dockerfile
FROM python:3.11-slim

WORKDIR /app

# Copy requirements first for better caching
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# Copy source code
COPY src/ ./src/
COPY artifacts/ ./artifacts/
COPY data/ ./data/

# Set Python path
ENV PYTHONPATH=/app/src

# Expose port
EXPOSE 8000

# Run the application
CMD ["uvicorn", "app.main:app", "--host", "0.0.0.0", "--port", "8000"]
```

## CI/CD Requirements

### GitHub Actions Workflow Needed
1. **Trigger**: Push to main branch
2. **Build Stage**:
   - Checkout code
   - Set up Python 3.11
   - Install dependencies
   - Run tests (scripts/test_*.py)
   - Build Docker image
3. **Push to Docker Hub**:
   - Login to Docker Hub
   - Tag image as `anasriad8/telco-churn-app:latest`
   - Push image
4. **Deploy to AWS**:
   - Update ECS service
   - Use AWS Fargate
   - Configure Application Load Balancer
   - Set environment variables

### Required Secrets
- `DOCKERHUB_USERNAME`
- `DOCKERHUB_TOKEN`
- `AWS_ACCESS_KEY_ID`
- `AWS_SECRET_ACCESS_KEY`
- `AWS_REGION` (e.g., us-east-1)
- `ECS_CLUSTER_NAME`
- `ECS_SERVICE_NAME`
- `ECS_TASK_DEFINITION`

### AWS Infrastructure (Already exists)
- ECS Cluster with Fargate
- Application Load Balancer
- Task Definition for the container
- Security groups and IAM roles
